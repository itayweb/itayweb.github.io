Welcome to my own website,
This website is about my hobby, and things to do.

My hobby is to develop things like: Games, Apps, Websites and more...
In the last 2 months I have been trying to develop a game.

At first, I started developing in Visual Studio By Microsoft in Monogame based on Xna which is a Framework (kind of game engine) and its written with C#.

Everything was fine, I made the character moving with animations, I made menu system, I made custom cursor.

But when I am trying to make the map, I just couldn't make it, I was searching in the Internet how to make it, but I couldn't find something working.

So here's some images of this game:


So I decided to leave this project and searching for another game engine to make the game.

So I found game engine called Unity that is very popular.

But also in this game engine I had a problem, the character was kind of sliding on the map.

So I decided to leave this project also and searching for another game engine.

So I found game engine called Unreal Engine 4 (or in short UE4) by Epic Games and written in C++ and unique language called UnrealScript, there are lots of titles which made with this game engine like: Fortnite, PlayerUnkown's Battlegrounds(PUBG), Gears Of War 4, and lots of more...

So far I made in this game engine player that can move with abillities, health and mana(skills usage bar) system, and some dummy enemies.

I am planning to make a world as big as I can, also I am planning to make quest system, enemy system, and more...
